## Installation

Use your favorite package manager to install this library from NPM.

```bash
# pnpm
pnpm add pawdirecte

# Yarn
yarn add pawdirecte

# npm
npm add pawdirecte

# Bun
bun add pawdirecte
```

You can find documentation with guides at [docs.literate.ink/pawdirecte](https://docs.literate.ink/pawdirecte).
